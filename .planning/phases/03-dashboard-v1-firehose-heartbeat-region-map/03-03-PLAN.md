<objective>
Implement the transport layer that every UI component in Plans 04–06 will depend on. Three deliverables:

1. Protocol types — three source files copied verbatim from the Grid (frame shapes, audit entry shape, region shapes) with SYNC headers so any future Grid-side change triggers a visible grep-hit.
2. WsClient — a class that owns the WebSocket lifecycle: full-jitter backoff reconnect, `lastSeenId` tracking, frame dispatch to registered handlers, and a bye-frame halt semantics.
3. Refill — a function that reacts to `dropped` frames by fetching the missing audit range from REST, with an in-flight guard (a second `dropped` during an active refill is ignored until the first completes) and an AbortSignal cleanup.

This plan is pure logic — no React, no DOM. Plans 04+ consume the WsClient.

    ```typescript
    /**
     * Full-jitter exponential backoff per AWS Architecture Blog.
     * delay = Math.random() * Math.min(MAX_DELAY_MS, BASE_MS * 2^attempt)
     * Range [0, ceiling) — 0 is allowed (immediate retry) per the full-jitter
     * definition; total thundering-herd avoidance relies on Math.random being
     * independent across clients.
     */
    export const MAX_DELAY_MS = 30_000;
    export const BASE_MS = 250;

    export function nextDelayMs(attempt: number): number {
        if (!Number.isInteger(attempt) || attempt < 0) {
            throw new TypeError(`attempt must be a non-negative integer, got ${attempt}`);
        }
        // Clamp 2^attempt to avoid Number overflow at extreme attempt counts.
        // Math.pow(2, 50) ≈ 1.1e15 — already far above MAX_DELAY_MS / BASE_MS,
        // so clamping at 50 is safe and cheap.
        const exp = Math.min(attempt, 50);
        const ceiling = Math.min(MAX_DELAY_MS, BASE_MS * (2 ** exp));
        return Math.random() * ceiling;
    }
    ```

    ```typescript
    export interface WsClientOptions {
        url: string;              // ws://localhost:8080/ws/events
        filters?: string[];       // glob patterns; undefined = all allowlisted
        wsFactory?: (url: string) => WebSocket;  // test seam — defaults to globalThis.WebSocket

    export type WsPhase = 'idle' | 'connecting' | 'open' | 'reconnecting' | 'halted' | 'closed';

    export interface WsClientState {
        phase: WsPhase;
        gridName: string | null;
        lastSeenId: number;
        reconnectAttempt: number;
    }

    export type WsEventMap = {
        hello: HelloFrame;
        event: AuditEntry;
        dropped: DroppedFrame;
        bye: ByeFrame;
        stateChange: WsClientState;
    };

    export class WsClient {
        constructor(opts: WsClientOptions);
        get state(): Readonly<WsClientState>;
        connect(): void;
        close(): void;  // intentional: halts reconnect, sends no-op, closes socket
        on<K extends keyof WsEventMap>(type: K, handler: (payload: WsEventMap[K]) => void): () => void;
        bumpLastSeenId(id: number): void;  // called by refill after successful fetch
    }
    ```

    Internal state machine:
    - `idle` → connect() → `connecting` (construct socket, set up listeners)
    - `connecting` → onopen → `open` (send subscribe frame)
    - `open` → onclose (bye NOT received) → `reconnecting` (schedule setTimeout with nextDelayMs)
    - `reconnecting` → timer fires → `connecting` (increment reconnectAttempt)
    - `open` → bye frame received → `halted` (set flag; next onclose does NOT schedule reconnect)
    - anywhere → close() → `closed` (tear down, clear timer, socket.close(1000))

    Frame dispatch in onmessage:
    - Parse JSON; if invalid → call onError, return
    - `hello` → update state.gridName + state.phase was 'connecting' → 'open' (already set in onopen)
    - `event` → update state.lastSeenId = entry.id (only if entry.id > current), fire handlers
    - `dropped` → fire handlers (do NOT mutate lastSeenId here)
    - `ping` → immediately socket.send(JSON.stringify({type:'pong', t: payload.t}))
    - `pong` → no-op (server-initiated heartbeat check)
    - `bye` → set halted flag, fire handlers, socket will close naturally

    Subscribe frame on every (re)connect after onopen:
    ```typescript
    socket.send(JSON.stringify({
        type: 'subscribe',
        ...(this.opts.filters && this.opts.filters.length > 0 ? { filters: this.opts.filters } : {}),
        sinceId: this.state.lastSeenId,
    }));
    ```

    Reconnect scheduling:
    - onclose (not halted, not intentional close): increment reconnectAttempt, compute nextDelayMs(attempt), setTimeout → call connect() again
    - On successful onopen: reset reconnectAttempt to 0

    ```typescript
    import type { AuditEntry } from '../protocol/audit-types';
    import type { DroppedFrame } from '../protocol/ws-protocol';

    export class RefillError extends Error {
        override readonly cause?: unknown;
        constructor(message: string, cause?: unknown) {
            super(message);
            this.name = 'RefillError';
            this.cause = cause;
        }
    }

    // Max page size matches grid/src/api/server.ts audit trail default behavior.
    const PAGE_LIMIT = 1000;

    // Module-level coalesce map: key is `${origin}|${sinceId}|${latestId}`.
    // A second call with the same key while the first is pending returns the
    // same Promise — prevents stampede when multiple components react to the
    // same DroppedFrame.
    const inFlight = new Map<string, Promise<AuditEntry[]>>();

    export async function refillFromDropped(
        frame: Pick<DroppedFrame, 'sinceId' | 'latestId'>,
        gridOrigin: string,
        onEntries: (entries: AuditEntry[]) => void,
        signal?: AbortSignal,
    ): Promise<AuditEntry[]> {
        if (frame.latestId === frame.sinceId) {
            onEntries([]);
            return [];
        }
        if (frame.latestId < frame.sinceId) {
            throw new RefillError(`latestId < sinceId (${frame.latestId} < ${frame.sinceId})`);
        }
        const key = `${gridOrigin}|${frame.sinceId}|${frame.latestId}`;
        const existing = inFlight.get(key);
        if (existing) return existing;

        const promise = (async () => {
            const collected: AuditEntry[] = [];
            let cursor = frame.sinceId;
            while (cursor < frame.latestId) {
                const remaining = frame.latestId - cursor;
                const limit = Math.min(PAGE_LIMIT, remaining);
                const url = `${gridOrigin}/api/v1/audit/trail?offset=${cursor}&limit=${limit}`;
                let response: Response;
                try {
                    response = await fetch(url, { signal });
                } catch (err) {
                    throw new RefillError(`Refill fetch failed at offset=${cursor}`, err);
                }
                if (!response.ok) {
                    throw new RefillError(`Refill HTTP ${response.status} at offset=${cursor}`);
                }
                const body = await response.json() as { entries: AuditEntry[]; total: number };
                if (!Array.isArray(body.entries)) {
                    throw new RefillError(`Refill response malformed at offset=${cursor}`);
                }
                collected.push(...body.entries);
                if (body.entries.length === 0) break;  // server exhausted
                cursor += body.entries.length;
            }
            onEntries(collected);
            return collected;
        })();

        inFlight.set(key, promise);
        promise.finally(() => { inFlight.delete(key); });
        return promise;
    }

    export function __resetRefillState(): void { inFlight.clear(); }
    ```
