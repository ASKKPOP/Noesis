    Handler sketch for `/api/v1/nous/:did/state`:
    ```typescript
    fastify.get<{ Params: { did: string } }>(
        '/api/v1/nous/:did/state',
        async (request, reply) => {
            const { did } = request.params;
            if (!DID_REGEX.test(did)) {
                return reply.code(400).send({ error: 'invalid_did' });
            }
            const runner = getRunner(did);
            if (!runner) {
                return reply.code(404).send({ error: 'unknown_nous' });
            }
            if (!runner.connected) {
                return reply.code(503).send({ error: 'brain_unavailable' });
            }
            try {
                const state = await runner.getState();
                return reply.code(200).send(state);
            } catch (err) {
                request.log.warn({ err, did }, 'brain get_state failed');
                return reply.code(503).send({ error: 'brain_unavailable' });
            }
        },
    );
    ```

        '/api/v1/economy/trades',

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| dashboard (browser) → grid /api/v1/* | Internet-adjacent HTTP; untrusted input |
| grid → brain (get_state RPC proxied through /api/v1/nous/:did/state) | Brain process is trusted but may be disconnected or throw |
| audit.query → /api/v1/economy/trades | Audit is trusted storage but entries may contain fields beyond the advertised shape |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-11 | Tampering | Path-param DID injection | mitigate | Strict `DID_REGEX` whitelist before any registry lookup; invalid returns 400 |
| T-04-12 | Information Disclosure | Brain RPC error message surfaced in 503 body | mitigate | Route catches the exception, logs server-side, returns fixed `{ error: 'brain_unavailable' }` |
| T-04-13 | Denial of Service | Unbounded `limit` on /economy/trades | mitigate | `Math.min(limitRaw, 100)` cap, integer coercion, NaN → default 20 |
| T-04-14 | Information Disclosure | Audit entry fields beyond `{counterparty, amount, nonce}` leaking to the client | mitigate | Route maps to a strict `TradeRecord` shape; test grep-asserts `Object.keys().sort()` equality |
| T-04-15 | Spoofing | Cross-origin POST abusing CORS | accept (D9) | Allowlist is frozen (localhost:3001/3000 only); no new origins added |
| T-04-16 | Elevation of Privilege | Consumers inferring write endpoints from read paths | mitigate | Only GETs added; 405 default on other verbs via Fastify default |
</threat_model>
