### Replay-mode frame (NEW, scoped to `/grid/replay`)

The replay viewer area (where `<Firehose>`, `<Inspector>`, `<RegionMap>` render in `replayMode`) carries a **2px amber border** to make the sandboxed state unmistakeable.

| Signal | Tailwind Token | Hex | Usage |
|--------|---------------|-----|-------|
| Replay frame border | `border-amber-400` / `border-2` | `#FBBF24` | Outer border on the viewer container div that wraps `<Firehose>`, `<Inspector>`, `<RegionMap>` in replay mode |

**Why amber:** amber-400 is the project's existing "caution" semantic color (used for `quorum_fail` in Phase 12, `operator.warned` events in Phases 6–7). It reads as "this is different from live" without destructive red.

### Accent reserved for (explicit list)

Brand accent (`#7DD3FC`) is reserved for:
1. `Replay` nav link text in the top nav (H3+ visible only)
2. Scrubber slider track fill (the filled portion of the `<input type="range">`)
3. Focus ring on tick jump input when focused
4. `Elevate` link text beside redacted fields in replay Inspector/Firehose
5. Any future tab underline if replay gets a sub-tab

Amber-400 (`--color-event-trade`: `#FBBF24`) is reserved for Phase 13 exclusively:
1. Replay frame border (2px outer border on viewer container)
2. `REPLAY` mode label badge

Amber MUST NOT appear on:
- Buttons or interactive controls (use accent or neutral)
- Page headings
- Error copy (use red/rose palette)

### Copywriting style guide (inherited from Phases 9–12)

- No exclamation marks. No emoji. No "Oops" or "Uh oh."
- Active voice: "Enter a tick range to begin inspection." not "A tick range should be entered."
- Named nouns: "Nous" (capital N), "Grid" (capital G), "tick" (lowercase), "Replay" (capital R as page proper noun).
- No temporal vague words: "tick 500" not "soon". Replay operates on ticks.
- "Read-only" hyphenated as compound adjective; standalone as predicate ("this view is read only").

```tsx
        Export audit chain slice
    </h2>
    <p id={warnId} data-testid="export-warning" className="mt-3 text-sm text-red-300 leading-relaxed">
        This export is permanent and cannot be undone. The tarball will contain the
        complete audit chain for the selected tick range. Anyone with the file can verify
        the chain.
    </p>
    <div className="mt-4">
        <div data-testid="export-grid-label" className="text-xs font-medium text-neutral-400">
            Grid to export
        </div>
        <code className="mt-1 block font-mono text-xs text-neutral-200 break-all">
            {capturedGridIdRef.current}
        </code>
    </div>
    <label className="mt-4 block text-sm text-neutral-200">
        <span data-testid="export-input-label">Type the Grid-ID exactly to confirm:</span>
        <input
            type="text"
            value={typed}
            onChange={(e) => setTyped(e.target.value)}
            onPaste={(e) => e.preventDefault()}
            onKeyDown={(e) => { if (e.key === 'Enter') e.preventDefault(); }}
            className="mt-1 w-full border border-neutral-700 bg-neutral-900 px-2 py-1 font-mono text-xs text-neutral-100 focus:border-neutral-500 focus:outline-none"
            data-testid="export-grid-input"
            autoComplete="off"
            spellCheck={false}
        />
    </label>
    {typed.length > 0 && (
        <p
            data-testid="export-hint"
            className={`mt-1 text-xs ${matched ? 'text-emerald-400' : 'text-neutral-400'}`}
        >
            {matched ? 'Match confirmed.' : 'Grid-ID does not match. Type exactly as shown.'}
        </p>
    )}
    <div className="mt-6 flex justify-end gap-3">
        <button
            type="button"
            autoFocus
            data-testid="export-cancel"
            onClick={() => dialogRef.current?.close()}
            aria-label="Keep chain slice private. No export will happen."
            className="px-3 py-1 text-sm text-neutral-200 hover:text-neutral-100"
        >
            Keep private
        </button>
        <button
            type="button"
            data-testid="export-confirm"
            onClick={onConfirm}
            disabled={!matched}
            aria-disabled={matched ? 'false' : 'true'}
            aria-label="Export this chain slice permanently. This action cannot be undone."
            className={
                matched
                    ? 'px-3 py-1 text-sm font-semibold text-red-400 hover:text-red-300'
                    : 'px-3 py-1 text-sm font-semibold text-red-900 cursor-not-allowed'
            }
        >
            Export forever
        </button>
    </div>
</dialog>
```

**Interaction contract:**
- Native `<dialog>` + `showModal()` (Phase 6 D-08 pattern, Phase 8 clone)
- `autoFocus` on "Keep private" (safer default — Enter on open cannot export)
- Paste suppressed on Grid-ID input: operator must type character-by-character
- Enter key blocked in Grid-ID input — confirm is always a deliberate click
- All close paths (ESC, backdrop click, "Keep private", programmatic) fire `onCancel`
- `onCancel` calls `agencyStore.setTier('H1')` (auto-downgrade H5→H1, D-13-07)
- Closure-capture: `capturedGridIdRef.current` snapshot at dialog-open time (Phase 6 D-07 / Phase 8 D-22 pattern)
- `onConfirm`: fires only when `typed === capturedGridIdRef.current`; performs POST to export endpoint; shows inline error if POST fails (dialog stays open)

## Interaction Contracts (State Machines)

### Replay page state machine

```
/grid/replay loaded
│
├──▶ tier === H1 | H2 → EmptyState "Replay requires H3 Partner or above."
│
└──▶ tier === H3 | H4 | H5
     │
     ├──▶ No tick range entered → EmptyState "No replay loaded." + scrubber (inputs empty)
     │
     └──▶ Tick range defined (startTick, endTick):
          │
          ├── Fetch chain slice for range → loading state (sr-only "Loading replay data…")
          │
          ├── Error → error copy from §Error States
          │
          └── Success → render scrubber (slider + inputs) + viewer frame
               │
               ├── Scrubber move → currentTick updates → viewer re-renders at tick N
               │
               ├── Jump to tick → validates in-range → currentTick = N
               │
               ├── Inspector click on Nous → selectedDid updates → Inspector re-fetches at tick N
               │
               ├── Telos field at tick N, tier < H4 → "— Requires H4" + Elevate link
               │    └── click Elevate → ElevationDialog(H4) → tier elevates → field reveals
               │
               ├── Whisper field at tick N, tier < H5 → "— Requires H5" + Elevate link
               │    └── click Elevate → ElevationDialog(H5) → tier elevates → field reveals
               │
               └── click "Export chain slice" (H3+)
                    └── ElevationDialog(H5) if tier < H5
                         └── on H5 confirmed → ExportConsentDialog opens
                              ├── type correct Grid-ID → "Export forever" enabled
                              │    └── click "Export forever" → POST /api/v1/replay/export
                              │         ├── success → dialog closes; operator.exported audit emitted
                              │         └── error → inline error in dialog; dialog stays open
                              └── "Keep private" / ESC / backdrop → dialog closes; tier auto-downgrade H5→H1

Route exit (navigate away from /grid/replay)
  └── useEffect cleanup → agencyStore.setTier('H1')
```

### Keyboard navigation

| Key | Context | Action |
|-----|---------|--------|
| Tab | Page loaded (H3+) | Cycle scrubber inputs → scrubber slider → Go button → Export button |
| Arrow keys | Scrubber slider focused | Move `currentTick` ±1 tick (native range slider behavior) |
| Enter | Go button focused | Apply jump-to-tick value |
| Tab | Viewer frame | Cycle Firehose → Inspector → RegionMap interactive affordances |
| Tab | Redaction placeholder focused | Reach `Elevate` link |
| Enter | `Elevate` link focused | Open ElevationDialog |
| Escape | ElevationDialog open | Close dialog |
| Escape | ExportConsentDialog open | Close dialog; auto-downgrade H5→H1 |
| Enter | ExportConsentDialog open | Blocked in Grid-ID input; no submit on Enter |
