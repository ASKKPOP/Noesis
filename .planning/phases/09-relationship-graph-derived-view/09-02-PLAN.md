Output: A self-contained `RelationshipListener` class under `grid/src/relationships/`, exported via barrel, with unit tests for every branch of the bump table and a grep gate ensuring no future file writes `edges.set/delete/clear`.

**`private handleEntry(entry: AuditEntry): void`:**
1. Dispatch on `entry.eventType`:
   - `'nous.spoke'` → extract `from_did`, `to_did` from `entry.payload` (match grid/src/dialogue/*.ts emitter keys exactly); call `this.applyBump(from_did, to_did, entry, this.config.bumpSpokeValence, this.config.bumpSpokeWeight)`.
   - `'trade.settled'` → extract the two participant DIDs from the trade payload (inspect economy producer to confirm key names, likely `buyer_did` + `seller_did` or `initiator_did` + `counterparty_did`); bump `bumpTradeSettledValence` + `bumpTradeSettledWeight`.
   - `'trade.reviewed'` → only fire on `verdict === 'fail'` OR whatever the reject discriminant is (confirm by reading the producer); extract the reviewed parties; bump `bumpTradeRejectedValence` (-0.10) + `bumpTradeRejectedWeight` (+0.05, because rejected still counts as contact per D-9-02 comment).
   - `'telos.refined'` → extract `nous_did` + `partner_did`; bump `bumpTelosRefinedValence` + `bumpTelosRefinedWeight`.
   - default: `return` (silently ignore all other allowlisted events — pure observer).

**`private applyBump(didA: string, didB: string, entry: AuditEntry, valenceBump: number, weightBump: number): void`:**
1. `let key: string; try { key = sortedPairKey(didA, didB); } catch { return; }` — D-9-11 self-loop silent-reject.
2. `const tick = entry.payload.tick as number;` — authoritative tick (D-9-12 forbids any other time source).
3. `const existing = this.edges.get(key);` — get-or-create edge.
4. If not existing: construct new Edge `{did_a: min(didA,didB), did_b: max(didA,didB), valence: 0, weight: 0, recency_tick: tick, last_event_hash: entry.eventHash}`.
5. Apply clamped bumps (RESEARCH.md lines 624-628 verbatim):
   ```
   edge.valence = Math.max(-1, Math.min(1, edge.valence + valenceBump));
   edge.weight  = Math.max( 0, Math.min(1, edge.weight  + weightBump));
   ```
6. `edge.recency_tick = tick;` `edge.last_event_hash = entry.eventHash;`
7. `this.edges.set(key, edge);` — the single Map write point in the entire codebase.

**Read API (no mutation):**
- `public getEdge(didA: string, didB: string): Readonly<Edge> | undefined` — computes `sortedPairKey` (catching self-loop → returns undefined), returns the Map entry or undefined.
- `public getTopNFor(did: string, n: number, currentTick: number): ReadonlyArray<{counterpartyDid: string; valence: number; weight: number; recency_tick: number; last_event_hash: string;}>`: iterate `this.edges.values()`, filter where `did_a === did` or `did_b === did`, decorate with decayed weight via `decayedWeight(edge, currentTick, this.config.tau)`, sort by decayed weight desc, take top `n` (caller clamps to `config.topNMax`).
- `public get size(): number { return this.edges.size; }` — for tests.
- `public allEdges(): IterableIterator<Edge> { return this.edges.values(); }` — read-only iterator (storage layer will consume this).

import { describe, it, expect } from 'vitest';
import { readdirSync, readFileSync, statSync } from 'fs';
import { join, relative } from 'path';

const GRID_SRC = join(__dirname, '../../src');

// Gate 1: In-memory edges Map mutations allowed ONLY in listener.ts (D-9-05).
const MAP_WRITE_PATTERN = /\b(?:this\.)?edges\.(?:set|delete|clear)\s*\(/;
const ALLOWED_MAP_WRITER = /relationships\/listener\.ts$/;

function walk(dir: string, files: string[] = []): string[] {
    for (const name of readdirSync(dir)) {
        const full = join(dir, name);
        if (statSync(full).isDirectory()) walk(full, files);
        else if (full.endsWith('.ts')) files.push(full);
    }
    return files;
}

describe('relationships producer boundary', () => {
    const all = walk(GRID_SRC);

    it('only listener.ts mutates the edges Map (D-9-05)', () => {
        const offenders: string[] = [];
        for (const f of all) {
            const src = readFileSync(f, 'utf-8');
            if (MAP_WRITE_PATTERN.test(src) && !ALLOWED_MAP_WRITER.test(f)) {
                offenders.push(relative(GRID_SRC, f));
            }
        }
        expect(offenders).toEqual([]);
    });

    // Plan 03 Wave 1 appends a second `it(...)` block for SQL-write boundary.
});
