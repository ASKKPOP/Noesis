Architecture

Frozen invariants

allowlist freeze-except-by-explicit-addition
27 events frozen. Every addition requires its own phase, closed-tuple payload test, sole-producer grep, and a literal bump in check-state-doc-sync.mjs.
AuditChain zero-diff invariant
Pure-observer listeners must not mutate the chain hash. Regression hash pinned since Phase 1 — any observer that touches the hash fails CI.
sole-producer boundary
One file per event type calls chain.append(). Grep CI gates enforce this — any stray append outside the designated producer file fails the build.
closed-tuple payload
Object.keys(payload).sort().join(',') === expected strict equality on every event. Extra or missing key is a contract violation caught in CI.
hash-only cross-boundary
Brain↔Grid wire carries hashes and bucketed levels only. Raw drive values, plaintext Telos, whisper plaintext never cross.
first-life promise (I-6)
Audit entries retained forever. Tombstoned DIDs permanently reserved — NousRegistry blocks DID reuse after bios.death.
chronos.* / rig.* prefix hard-ban
Permanently banned from the production allowlist. check-state-doc-sync.mjs enforces via checkChronosPrefixBan() and checkRigPrefixBan() — Phase 14 D-14-08.
replay.* prefix hard-ban
ReplayGrid runs its own isolated chain. replay.* names permanently banned from the production allowlist — Phase 13.
rigs are configured production code, not forks
scripts/rig.mjs wraps GenesisLauncher unchanged. CI bans httpServer.listen, wsHub, and all bypass flags from rig entry files.
DID regex
/^did:noesis:[a-z0-9_\-]+$/i enforced at all entry points. Tombstone check at 4+ route handlers. GDPR-style erasure remains out of scope.
